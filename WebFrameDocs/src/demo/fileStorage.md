# 文件对象存储

源码地址：`WebFrameDocs/src/demo`  下  `fileStorage`

### 一、使用方式

1. 安装依赖
   
    ```shell
   pip install -r requirements.txt
   ```
2. 启动实例
   
    ```shell
    python3 app.py
    ```
   
3. 查看api文档

    ```shell
    http://127.0.0.1:8000/docs
    ```

### 二、目前功能

1. 单文件上传
2. 多文件上传
3. 文件下载
4. 文件删除
